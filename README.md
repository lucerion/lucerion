## Vim plugins

* [vim-i18n-rails ](https://github.com/lucerion/vim-i18n-rails) - Rails I18n Vim plugin
* [vim-rubygems](https://github.com/lucerion/vim-rubygems) - Ruby `gem` package manager Vim plugin
